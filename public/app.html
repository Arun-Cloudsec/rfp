Response Quality Scores
How scores are calculated
📋 Requirements Coverage — % of stated requirements that are explicitly addressed in the response. Higher = fewer gaps.
🔧 Technical Alignment — How well the proposed solution matches the technical specifications. Checks for specific tools, platforms, and methodologies.
✅ Compliance Match — How well the response addresses regulatory and standards requirements (ISO, SOC 2, GDPR, etc.).
📝 Clarity & Structure — Readability, logical flow, use of headings, and professional tone.
🏆 Win Probability — Composite score (weighted average of the four dimensions above) estimating likelihood of winning the bid. 70%+ = strong, 50–69% = moderate, <50% = needs improvement.
⚡ Scores are AI-estimated based on content analysis — use them as a guide, not a guarantee.

1
Configure the engagement — select deployment type, duration, and describe the project scope in detail.

Azure Billing Agent
💳 Azure Billing Agent — How It Works
1
Connect Azure — go to Settings → Azure Billing Credentials and enter your Tenant ID, Client ID, and Client Secret. You need Cost Management Reader role assigned to your App Registration.
2
Select subscription — use the dropdown at the top to pick which Azure subscription to analyse.
3
View cost breakdown — see costs by subscription, resource group, and top billed resources for the current month.
4
Find idle resources — the agent scans for unused VMs, empty storage accounts, and unattached disks that can be deleted to save money.
5
AI recommendations — ask the AI chat to explain costs, suggest optimisations, or compare month-on-month spending.
💡 Tip: Assign the Cost Management Reader role at Management Group level so it covers all subscriptions automatically.
